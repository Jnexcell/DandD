# Monsters by Challenge Rating

Monsters grouped by Challenge Rating (ascending), each as **Name (CR, printed page)**. Use it to pick threats for an encounter at a glance, then total XP with the table in `README.md`.

**Source:** Monster Manual stat blocks (`Challenge` lines), printed pp.12–350. Page-offset reminder: physical PDF page = printed page + 1.

> **Completeness:** **CR 0–10 is built for completeness** (the practical one-shot range), drawing from both the A–Z section and Appendix A (beasts) / Appendix B (NPCs). **Above CR 10** lists notable/commonly-used monsters only — for anything else use `monster-index.md` to find the page, then read the PDF stat block. Where a group shares a page (dragons, demons, giants), the page is the group/age-category page.
