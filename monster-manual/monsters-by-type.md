# Monsters by Creature Type

Monsters grouped by the 14 creature **types**, each as **Name (CR, printed page)**. Use it when an effect targets a type (e.g. *protection from evil and good* vs. aberrations/fiends/undead, an *arrow of dragon slaying*, *turn undead*), or to browse for a themed encounter.

**Source:** Monster Manual stat block size/type lines and `Challenge` lines (printed pp.12–350); type descriptions MM pp.6–7. Page-offset reminder: physical PDF page = printed page + 1.

> **Representative, not exhaustive.** This favors commonly-used monsters and is most complete in the low–mid CR range. For a guaranteed-complete name list use `monster-index.md`; for full CR coverage use `monsters-by-cr.md`. A monster's **(tag)** — e.g. *(demon)*, *(goblinoid)*, *(shapechanger)* — is noted where useful but carries no rules of its own. The book's own **Index of Stat Blocks** (p.351) lists every individual stat block.
