# Spell Index (All PHB Spells)

Master alphabetical index of every spell in the Player's Handbook. Use it to find a spell's level, school, which classes can cast it, and its page. **Full mechanics** live in the per-level files: cantrips in `cantrips.md`, 1st-level in `level-1.md`, and (when built) `level-N.md` for higher levels.

**Source:** PHB Chapter 11, "Spells." Class associations are from the class spell lists (printed p.207–210); level and school are from each spell's description header (printed p.211–289). **Level 0 = cantrip.** Class abbreviations: **Bard (Brd), Cleric (Clr), Druid (Drd), Paladin (Pal), Ranger (Rgr), Sorcerer (Sor), Warlock (Wlk), Wizard (Wiz)**. "(ritual)" marks spells that can also be cast as rituals.

> Note: spell names and page numbers are reconciled against the canonical 2014 PHB; the source text dump's spell lists were heavily column-scrambled by OCR, so class associations were taken from the standard PHB class lists. A handful of class associations that are easy to mis-key are flagged for verification.

## Notes on naming

- Spells named after a wizard (Leomund, Mordenkainen, Otiluke, Otto, Melf, Tenser, Bigby, Evard, Rary) are alphabetized in the PHB under both the wizard's name and (for several) a common name. This index lists the **primary PHB entry**; common-name aliases are cross-listed and point to the same page (e.g., *Floating Disk* → *Tenser's Floating Disk*, p.282; *Faithful Hound* / *Private Sanctum* are *Mordenkainen's…*, p.257).
- Page numbers are the **printed** PHB pages (the PDF's physical pages run roughly 19–24 lower across this range; see the page-offset note in the project `CLAUDE.md`).
